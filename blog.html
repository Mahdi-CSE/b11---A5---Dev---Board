Question-1: What are the different ways to select an element in the DOM?

There are several methods to select elements in the DOM using JavaScript.
1.getElementById.
2.getElementsByClassName
3.getElementsByTagName
4.querySelector
5.querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText, and textContent are used to get content from DOM elements in JavaScript.
innerHTML:Gives the entire HTML structure inside the element, including tags, attributes, and text.
innerText: Gives the content inside the tags like p tag, header tag etc.
textContent: Only gives the text elements inside the tag including the hiddentext.

Question-3: What is event delegation in the DOM?

Instead of adding multiple event listener to a element, with this technique we add the event function to its parent element and use the child elements to handle those event.

Question-4: What is event bubbling in the DOM?

If there are multiple event listener on its parent nodes those events trigger after their child nodes if their trigger condition is met. This is called Event Bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating Elements:We can create new elements using the createElement() method.
Adding Elements:We can add new elements using the appendChild() method.
Removing Elements:We can remove elements using the remove() method.
